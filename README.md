# NS

NS 是一个复杂网络模型实验项目，用 Python 生成不同类型的网络，并计算或绘制网络结构特征。

项目包含四类网络实验：

| 文件 | 内容 |
|---|---|
| `ba.py` | 生成 BA 无标度网络，展示优先连接机制形成的网络结构。 |
| `gnm.py` | 生成 G(n,m) 随机图，观察固定边数随机连接下的度分布。 |
| `gnp.py` | 生成 G(n,p) 随机图，统计平均度、平均聚类系数和平均路径长度，并绘制度分布。 |
| `ws.py` | 生成 WS 小世界网络，统计平均度、平均聚类系数和平均路径长度，并绘制网络结构和度分布。 |

项目主要用于对比随机网络、小世界网络和无标度网络在结构特征上的差异，包括度分布、聚类特征和路径长度。
